Hi, I'm Xyste Chrysologue Rabearson

I’m a passionate frontend developer creating clean, responsive websites using HTML, CSS, and JavaScript. I’m currently studying Software Development at BYU-Idaho.

My Journey into Web Development

My journey in web development started when I worked with a friend who was learning Python. Intrigued by his progress, I decided to start learning alongside him. However, I quickly realized that self-training was more challenging than I anticipated. I wanted a more structured path, so I searched for affordable courses that could guide me. That's when I discovered the BYU Pathway program, an incredible program offering certificates in various fields. I chose the Software Development path, which required completion of the Web and Computer Programming Certificate and the Web Development Certificate. This is where my real journey into web development began, and I’ve been passionately working towards my goal ever since.

My Goals

My ultimate goal is to become a software developer who solves real problems and makes people's lives easier through innovative solutions. I’m particularly interested in building systems that improve everyday experiences, whether through web applications, software tools, or other technological solutions.

Current Focus
